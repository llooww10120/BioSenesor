# BioSenesor

### 各程式用途

#### wettest_point_csv_to_picture.py

> 將所有時間中最濕的點畫出，用來和慈濟的試紙攝影做比對

#### moist_resistor_in_board_for_each_urine_point.py

> 畫出各尿點隨時間變化的曲線，尿點座標取自urinepoint.py

#### urinepoint.py

> 尿點座標

#### senserlist.py

> 濕敏電阻排列座標

#### moist_resistor_in_board.py

> 畫出全版隨時間濕度變化曲線

#### csv_to_video.py

> 讀取csv檔產生video
